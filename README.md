# GraphQL API with FastAPI

This project implements a simple GraphQL API using FastAPI.

## Setup

1.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
    (Note: You may need to create a `requirements.txt` if not already present.)

2.  **Run the application:**
    ```bash
    uvicorn main:app --reload
    ```

The API should then be accessible at `http://127.0.0.1:8000/graphql`.
